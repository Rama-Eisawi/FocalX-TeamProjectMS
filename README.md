# FocalX-TeamProjectMS
## Overview
This project is a Project Management System built with Laravel that allows users to manage projects, tasks, and roles. It supports multiple user roles (Manager, Developer, Tester) and provides an interface for managing tasks in various projects. The system includes features such as task assignment, role management, and project tracking.
